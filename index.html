AI Image to Text Application

This real-world application is designed for a factory that receives a single, large PDF file from an industrial scanner containing hundreds of scanned delivery order images with no searchable text—making it tedious to manually locate a specific order.

The solution automatically splits the PDF into individual pages and uses Python AI-powered OCR technology to detect the invoice number on each page. It then renames the file accordingly for quick retrieval.
